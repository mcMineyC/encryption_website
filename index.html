Pi Cipher

How it works: Add the digits of Pi to your letters! In the example message, "Hello World!", 3 will be added to H, making it K. 1 is added to e, making it f. Add 4 to l and get p, add 1 to l and get m, etc.

Stats:

  • Type: Vigenère cipher (but with an infinite keyword!)
  • Accepted characters: all (non-letters remain unchanged)
  • Case sensitive: yes
  • Uses randomness in encryption: no

keyboard Shift Cipher

How it works: To encrypt a message with this cipher on your keyboard, simply shift your fingers over one key to the right and start typing! The rightmost keys wrap around. "Qwerty" becomes "Wertyu" and so forth.

Stats:

  • Type: Substitution cipher
  • Accepted characters: all (non-qwerty characters remain unchanged)
  • Case sensitive: yes
  • Uses randomness in encryption: no

ROT13

How it works: ROT13 is a Caesar cipher that replaces each letter with the 13th letter after it in the alphabet. "A" becomes "N", b" becomes "o", and so forth. Note that ROT13 is its own inverse.

Stats:

  • Type: Caesar cipher
  • Accepted characters: all (non-letters remain unchanged)
  • Case sensitive: yes
  • Uses randomness in encryption: no
